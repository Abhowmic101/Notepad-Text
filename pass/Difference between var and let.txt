Variables defined with var are hoisted to the top and can be initialized at any time.

Meaning: You can use the variable before it is declared

Redeclaring a JavaScript variable with var is allowed 
anywhere in a program.

Redeclaring a variable using the var keyword can impose 
problems.

Redeclaring a variable inside a block will also redeclare
the variable outside the block.
Variables declared with the var keyword can NOT have 
block scope.

Variables declared inside a { } block can be accessed 
from outside the block.

Redeclaring a variable with let, in another block, IS
allowed.

With let, redeclaring a variable in the same block is 
NOT allowed.
