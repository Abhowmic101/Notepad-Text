A tag for an element without a closing tag is known as a self-closing tag.

figure tag or  The figure element represents 
self-contained content and will allow you to associate an
image with a caption.
A figure caption (figcaption) element is used to add a 
caption to describe the image contained within the figure
element.

The input element allows you several ways to collect data
from a web form. Like img elements, input elements are 
self-closing and do not need closing tags.

abel elements are used to help associate the text for 
an input element with the input element itself 
(especially for assistive technologies like screen reader
s).

The fieldset element is used to group related inputs and
 labels together in a web form. fieldset elements are 
block-level elements, meaning that they appear on a new 
line. 

The legend element acts as a caption for the content in
 the fieldset element. It gives users context about what
 they should enter into that part of the form.
Your legend element should be the first element right 
below fieldset element's opening tag and before the first
 radio button's opening label tag. It is not in the 
correct position.

meta elements are self-closing.

article elements commonly contain multiple elements that
have related information.
