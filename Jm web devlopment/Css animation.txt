transform property is use to transform an html element.
if you use same css property more than once in one css rule then
then according to cascading then downward most css is used.
 like:

div{
transform:scale(.6);
transform:rotate(45deg);
}
the last one will be used. To use both you can use them
togther using a space between the values.
like:
div{
transform:scale(.6) rotate(45deg);
}
